# Rerisk
python app that will eventually power a flask website that will allow users to check common risk measurements for particular stocks or their portfolio 
## yfinance, datetime, numpy, pandas, scipy.stats
please install all of the above packages. there is a venv package being added to simplify this. 
## methods
bottom 5 will work out the daily returns over the selected period and then return the average over the bottom 5%
normal_dist will check the data fits a normal distribution and then returns the 99th and 95th losses
